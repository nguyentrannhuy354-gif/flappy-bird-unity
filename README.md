A simple Flappy Bird clone developed with Unity 6 and C#.

## Features

- Endless gameplay
- Random pipe generation
- Score system
- Collision detection
- Game over and restart
- Custom hand-drawn 2D sprites

## Technologies

- Unity 6
- C#
- Unity Input System

## How to Play

- Press Space or Left Mouse Button to jump.
- Avoid the pipes and try to get the highest score.
